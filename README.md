# GenMixture

**GenMixture** is an open-source orchestration layer for **evaluation-driven mixing and selection of outputs from multiple generative models**, including LLMs, diffusion models, and multimodal systems. It treats inference as a mixture problem, where multiple candidate outputs are generated, scored, and composed into a single optimized response using explicit and configurable strategies.

GenMixture is model-agnostic and production-minded, enabling routing, ranking, and composition across heterogeneous generators with full transparency and reproducibility. It is designed for ensemble generation, provider arbitration, and research on mixture-based inference in modern generative systems.
